# 语音助手 (RedMagicVoice) v 1.0.0

一款专为高性能 Android 设备打造的专业离线语音助手，支持实时录音转文字及批量音频文件转录。

## 核心特性
- **离线识别**：采用 Vosk 引擎，无需联网即可实现高精度中文识别，保护隐私。
- **实时录入**：毫秒级响应，支持自动去空格及智能分段。
- **批量转录**：支持多线程并行处理多个音频文件（WAV/MP3/M4A），转录效率翻倍。
- **自动保存**：转录结果自动以同名形式保存至手机“下载 (Download)”文件夹。
- **手动编辑**：复古书页风格编辑器，支持实时校对、一键复制及导出 TXT。
- **精致 UI**：动态声波波纹动画，深灰色简洁交互按钮。

## 系统要求
- **操作系统**：Android 14.0 (API 34) 及以上。
- **硬件**：建议使用 4GB 内存以上的高性能移动设备。

## 使用说明
1. 授予录音权限以启动引擎。
2. 点击“开始识别”进行实时录音转写。
3. 点击“批量选取音频”处理本地存量录音文件。
4. 转录完成后可在右下角点击“查看保存文件”直接定位结果。
